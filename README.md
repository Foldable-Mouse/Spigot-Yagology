# Spigot-Yagology
A Minecraft plugin that adds several custom items, each with their own unique crafting recipe and qualities. 
This plugin falls under the theme of Minecraft player "Yaganese" (https://namemc.com/profile/Yaganese.1).
To get started, add this plugin to your server and use /yagrecipes (/yr for short) to view all items featured. 
Feel free to use this plugin for any purpose :)
